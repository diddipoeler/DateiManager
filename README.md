# DateiManager

Moderner PHP-Dateimanager ohne Framework.

## Version

v0.1.0 – Grundsystem

## Voraussetzungen

- PHP 8.2 oder höher
- Composer

## Installation

```bash
composer install
php -S localhost:8000 -t public
```

Danach im Browser öffnen:

```text
http://localhost:8000
```

## Enthalten in v0.1.0

- PSR-4 Autoloading
- MVC-Light Grundstruktur
- Bootstrap 5 Oberfläche
- Request-/Response-Klassen
- View-System
- Grundlayout
- vorbereitet für Dateinavigation
